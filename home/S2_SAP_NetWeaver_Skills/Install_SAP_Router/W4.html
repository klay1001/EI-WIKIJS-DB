長華 SAP Router (Linux) 安裝及管理說明

 

Linux Server IP:

172.31.42.139

 

SAP Router安裝路徑：

/usr/usr/sap/saprouter 目錄
à saprouter 主程式

à niping 網路工具

à saprouttab 存取權限設定檔

à dev_rout 程式 trace file

à saprouttab.log 存取記錄

 

Linux OS User :

saprouter 帳號，專門用來執行 saprouter.

 

SAP Router 自動啟動之 RC scripts:

/etc/rc.d/saprouter

/etc/rc5.d/S90saprouter à ../saprouter

/etc/rc5.d/K90saprouter à ../saprouter
