結論 (For Production System only)：

 

HANA Server SAPS 值

l  Total SAPS(DB+APP) 為 12000, 建議用 16000  (因 sizing tool 不考慮客製，以1.5x 解決)。

 

HANA Server Memory:

l  HANA Database server 338GB。因要裝上 Application Server ，會再使用 16GB左右，故將取 512GB level 的 HANA System

 

HANA Server Disk:

l  DATA (/hana/data) 空間

n   以 HANA DB Memory 4 倍來計， 338GB x 4 = 1352GB

l  LOG (/hana/log) 空間

n   以 HANA DB Memory 1 倍來計，338GB x 1 = 338GB

l  OS (/) 空間至少 100GB

 

Hardware 採購注意事項

l  Production SAP HANA Server之機型需符合以下二項條件之一

n   Certified HANA Appliance
http://global.sap.com/community/ebook/2014-09-02-hana-hardware/enEN/appliances.html

n   SAP TDI (Tailored Data Center Integration) Program 所明列的 Entry-Level HANA E5 Systems之一
http://global.sap.com/community/ebook/2014-09-02-hana-hardware/enEN/entry-level-systems.html
並需通過 SAP Hardware Configuration Check Tool (HWCCT) 之檢驗 (SAP note 1943937)
