增加SWAP空間

Suse linux swap
假設要加 swap 8192MB 在 /var/lib/swap 目錄裡

1. Create empty swap file

mkdir -p /var/lib/swap
dd if=/dev/zero of=/var/lib/swap/swapfile bs=1M count=16384
(Oracle 11.0.2.4 需求16GB)
2. Initialize this swap file

mkswap /var/lib/swap/swapfile

3. Active swap

swapon /var/lib/swap/swapfile
4. 寫在 fstab 檔使重啟後仍生效

在 /etc/fstab 加一行
/var/lib/swap/swapfile swap swap defaults 0 0
