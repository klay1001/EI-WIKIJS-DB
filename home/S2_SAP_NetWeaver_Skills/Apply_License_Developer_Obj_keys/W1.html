如何申請 SAP License

準備

 

l  首先您要有一組有效的 SAP Support Portal S-User 帳號，具有申請 License 的功能。

l  您還需要欲申請系統的 Active Hardware Key。用 T-Code SLICENSE 可以取得。

 

申請步驟

 

請用Browser 連線到 https://support.sap.com/licensekey  ，如下圖

按上圖中的 “Launch the License Key Application” 方塊

 

 

 

 

 

 

 

 

 

 

出現登入畫面，請打入您的 S-User 帳號密碼。

再按 Log On 完成登入。會進入以下畫面.

 

在該畫面裡，請選取您的 customer 名稱，或利用 Installation Number 去尋找。

上圖中，是利用 Installation Number 欄位去選取您的系統所屬的 Installation Number。如上例是選 ERP – T installation 這一項。

 

畫面接下來會列出在此 installation number 裡已申請過的系統。若您要新申請 License，請按右上方的 圖示。進入下面畫面。

若您的系統是 ERP6.0 EHPx ，請選用 “SAP ERP”，若是 S/4HANA，請選用 SAP S/4HANA。若是 Java based ，通常選用 SAP NetWeaver。

再來選擇版本，本例為 ERP 6.0，完選擇後，系統會進入如下畫面。

上圖中，請在各欄位選用您的系統所對應的資訊。以本例來看，System ID 是 IDS, 是 Oracle Database 及 Windows 作業系統。選擇完成後，按 Continue。

接著進入上圖，我們要加入系統，

註： 請在原系統利用 T-Code SLICENSE 查詢該系統的 Hardware Key。如下圖中的 “Active Hardware Key” 一欄。

按 “Add License Key” 圖示。

請選擇 License Type。一般若是 ABAP Based 的 ERP 系統，請選用 Standard – Web Application Server ABAP or ABAP+JAVA。若是 EP 7.0 的，請選 J2EE-Engine。選擇完後進入下圖。

接下來填入您的 Hardware Key。最後一欄 “Valid until” 請保留為 31-12-9999，表示永久的 license。然後按 Add。

 

接下來會有產生二個 license entry，一個是永久的(Permanent) ，另一個是每三個月要更新的 Maintenance License。

 

然後我們要系統狀 key 值產生，我們才能 download file 用 SLICENSE 匯入系統，請按右下的

 

系統畫面會經過幾次 refresh，然後到下面的畫面。

上圖表示已經產生了二個 Digital Signed License key。請勾選上面二項，再按右上的 去下載檔案。

到此申請 License 的動作就完成了。
